# 我的个人博客（Hexo）

这里是我的个人博客项目，基于 [Hexo](https://hexo.io/) 搭建，用来记录：

- 学习与算法笔记
- 研究方向与论文阅读心得
- 日常随记与生活感想

文章源码主要位于 `source/_posts/` 目录，Hexo 生成的静态页面会输出到 `public/` 目录。线上部署时，会将 `public/` 内容发布到实际博客站点。

仓库中包含：

- Hexo 博客的配置（如 `_config.yml`）
- 页面与主题相关资源（例如 `themes/`、`source/about/` 等）
- 部分基础部署脚本与配置（例如用于本地预览/部署的 `Dockerfile`）

实际文章内容目录（如 `source/_posts/`）在 Git 提交时会被忽略，用于保护个人写作内容与隐私。
